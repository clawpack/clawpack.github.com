
.. _igetsp:

============
``igetsp``
============

Signature: 
	``integer function igetsp(nwords)``


Arguments: 
	``integer, intent(in):``
		* ``nwords``: Length of continuous space (double 
		  precision) to be allocated.


Description: 
	Allocates contiguous space of length ``nwords`` in the main 
	storage array ``alloc``.
