About PyClaw¶

PyClaw is an open-source, free project. If you find PyClaw useful, please let us know via a short e-mail to any or all of the following:

  • mandli@uw.edu
  • david.ketcheson@kaust.edu.sa

Citing¶

If you use PyClaw in work that is published, please cite

Kyle Mandli, David I. Ketcheson, Amal Alghamdi, and Aron Ahmadia, PyClaw Software <version number>

Contributing¶

Contributions to the package are most welcome. If you have used PyClaw for research, chances are that others would find your code useful. See Information for developers for more details.
