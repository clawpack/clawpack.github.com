===================
Pyclaw Data Objects
===================

All Pyclaw data objects inherit from the basic functionality of :class:`Data`.
It defines the way all data files are read and written and provides
functionality for doing both.  Please see :class:`Data` for more information
for general functionality and each individual definition for the specifics.
