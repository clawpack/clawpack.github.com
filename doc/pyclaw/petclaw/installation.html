Installation with 64-Bit Enthought Python and PETSc-dev (+ petsc4py-dev)¶

The development versions of PETSc and petsc4py are needed to use the implicit time stepping in PETCLAW.

Platform¶

  • Mac OS X 10.6.6 (Intel Core i5)

Install Enthought Python distribution 64-bit¶

Download the 64-bit enthought python distribution from: http://download.enthought.com/academic-epd-7.0/

At the time of writing this document the most recent stable release is epd-7.0-2-macosx-x86_64.dmg

  • Double-click on the .dmg file to mount the image
  • Double-click on the .mpkg file to run the installer

Numpy¶

The latest known working version is 1.5.1 which is included in 64-bit enthought python epd-7.0-2-macosx-x86_64.dmg. On 2011-05-14 numpy 1.6 has been released. This release has not been tested yet!

To test the numpy 1.5.1 functionality open a terminal and run python:

$ python

Then do:

>>> import numpy
>>> numpy.test()
You should get something like:
  • Ran 2983 tests in 10.194s
  • OK (KNOWNFAIL=4, SKIP=1) <nose.result.TextTestResult run=2983 errors=0 failures=0>

Install PETSc-dev¶

The development version of PETSc can also be obtained using three approaches. Here we use Mercurial to get it (recommended by PETSc web-page). Look at http://www.mcs.anl.gov/petsc/petsc-as/developers/index.html for more information.

With Mercurial do

$ cd path/to/the/dir/where/you/want/download/Petsc-dev
$ hg clone http://petsc.cs.iit.edu/petsc/petsc-dev
$ cd petsc-dev/config
$ hg clone http://petsc.cs.iit.edu/petsc/BuildSystem BuildSystem

Add the following lines to you .bash_profile (sh, bash, or zsh shell)

$ export PETSC_DIR=path/to/the/dir/where/you/downloaded/Petsc-dev/petsc-dev
$ export PETSC_ARCH=arch-osx-10.6

whereas if you are using a csh/tcsh shell add the folliwing lines to your shell start-up file

$ setenv PETSC_DIR path/to/the/dir/where/you/downloaded/Petsc-dev/petsc-dev
$ setenv PETSC_ARCH arch-osx-10.6

Then, configure PETSc-dev as usual, i.e.

$ /config/configure.py --with-cc=gcc --with-fc=0 --with-cxx=g++ --with-python=1 --download-mpich=1 --with-shared-libraries=1
