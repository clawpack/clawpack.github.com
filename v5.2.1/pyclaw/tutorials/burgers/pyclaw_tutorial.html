PyClaw Tutorial¶

PyClaw Tutorial by Aron Ahmadia is licensed under a Creative Commons Attribution 3.0 Unported License.

Import numpy, matplotlib, and clawpack packages¶

[1]:

import numpy as np
import matplotlib as mpl
import matplotlib.pyplot as plt
from clawpack import riemann
from clawpack import pyclaw

Instantiate solver¶

[2]:

solver = pyclaw.ClawSolver1D()
solver.kernel_language = 'Fortran'

Choose a limiter¶

[3]:

solver.limiters = pyclaw.limiters.tvd.vanleer

Choose appropriate Riemann solver¶

[4]:

solver.rp = riemann.rp1_burgers
solver.num_waves = 1
