Recent changes — release notes¶

Clawpack 5.x¶

  • Click on the Install tab or visit Installing Clawpack for installation instructions.
  • Versions of Clawpack 5.0 can be found at https://github.com/clawpack/clawpack/releases/.
  • Recent versions can also be found on Zenodo with permanent DOIs, listed below.
  • Please see Citing this work for the suggested way to cite the version of Clawpack that you are using in publications.
  • Clawpack 5.0 has significant changes from past versions. See Changes in Clawpack 5.0.
  • For changes in PyClaw, see the PyClaw changelog.
