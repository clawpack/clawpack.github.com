Clawpack Gallery of Jupyter Notebooks¶

The Jupyter notebook (formerly known as IPython notebook) is a very nice platform for illustrating Clawpack examples.

If you have used Clawpack with the Jupyter notebook, please send us a link or submit a pull request to the apps repository. The links below will take you to static view of several notebooks as html files. You can also play animations in them and interact with some plots, but to actually run the code yourself you should clone the apps repository.

Warning: The notebooks and tools to support them are being actively developed, so these may change in the near future and the versions posted here may not work with the current Clawpack.

Examples using PyClaw:

  • A quick introduction to PyClaw
  • A 2D fluid dynamics example
  • Stegotons: solitary waves arising in non-dispersive periodic media
  • Demonstration of different limiters for advection
  • PyClaw’s Geometry tools
  • Solitary waves formed by diffraction of shallow water waves over a corrugated bottom

Examples using Fortran Classic:

  • advection_1d illustrating upwind, Lax-Wendroff, and limiter methods.
  • acoustics_1d_example1

Examples using AMRClaw:

  • advection_2d

Examples using GeoClaw:

  • chile2010a illustrates how to set up a basic GeoClaw run with adaptive refinement.
  • chile2010b illustrates setting regions and gauges.
  • topotools_examples illustrates some of the tools from topotools module for working with topography data.
  • dtopotools_examples illustrates some of the tools from dtopotools module for moving topography.
  • Okada illustrates use of the Okada model for generating sea floor deformation.

Examples for VisClaw:

  • Animation tools demo

Riemann solvers:

A set of notebooks is under development to illustrate Riemann solvers. See http://www.clawpack.org/riemann_book/index.html.
